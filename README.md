# Ojas

A selfcare website that will enhance a studentâ€™s overall health.The application is designed to provide users with resources and tools that can help them improve their overall health and well-being.
The website comprises of three sections :

- Physical Section : It includes a BMI page that will calculate the BMI of the students and based on the result we recommend them exercises and diet plan.
- Mental Section: This section encompasses various activities like meditation, painting, soothing music and voice notes to help the students calm their minds and become joyful.
- Social Section: Discord server that consists of various channels which will help the students communicate with each other.

## Features

The application includes the following features:

### Mental Well-being

- Meditation exercises: A library of guided meditations that users can follow to reduce stress and improve mental clarity.
- Journaling: A feature that allows users to create journal entries to track their thoughts and emotions over time.
- Mindfulness exercises: A set of exercises designed to help users become more mindful and present in their daily lives.

### Physical Well-being

- Exercise tracker: A feature that allows users to create and track exercise routines to improve their physical health.
- Nutrition tracker: A tool that enables users to track their daily calorie intake and monitor their diet to improve their physical health.
- Sleep tracker: A feature that allows users to track their sleep patterns to ensure they are getting enough rest.

### Social Well-being

- Community groups: A feature that enables users to join groups and connect with others who share similar interests and goals(through Discord).
- Events calendar: A tool that allows users to find events in their area that promote social well-being, such as networking events or social clubs.

## Installation

To install the application, follow these steps:

1. Clone the repository to your local machine.
2. Install the required Python packages using pip install -r requirements.txt.
3. Set up a local database using python manage.py migrate.
4. Run the development server using python manage.py runserver.

## Usage

To use the application, navigate to the local URL provided by the development server. From there, you can create an account and start exploring the different features of the app.
